# Six Degrees of Football: Network Science of Player Transfers in Europe's Top-5 Leagues

This project analyzes football player transfers across Europe's top five leagues (Premier League, La Liga, Serie A, Bundesliga, and Ligue 1) using network science techniques.

## Overview

A player transfer network was constructed from more than 46,000 transfer records spanning 1992–2024. Clubs are represented as nodes and player transfers as weighted edges.

The project investigates:

- Network connectivity and small-world properties
- Centrality measures
- Community detection
- K-core decomposition
- Assortativity
- Temporal network evolution
- Effects of the 1995 Bosman ruling
- Link prediction using machine learning

## Dataset

- 46,000+ transfer records
- 19,000+ players
- Top 5 European football leagues
- Period: 1992–2024

## Methods

### Network Analysis
- Degree Centrality
- Betweenness Centrality
- Closeness Centrality
- Eigenvector Centrality
- Community Detection (Louvain)
- K-Core Decomposition
- Small-World Analysis

### Machine Learning
- Graph-based Feature Engineering
- Logistic Regression
- Link Prediction

## Tools & Technologies

- Python
- NetworkX
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Results

The analysis reveals strong small-world characteristics, highly connected transfer hubs, and distinct league-based communities. Temporal analysis also demonstrates significant structural changes following the 1995 Bosman ruling.
